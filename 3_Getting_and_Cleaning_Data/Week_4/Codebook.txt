The output contains  the average of each variable for each activity and each subject. It is a data frame with each column and row corresponding to a variable and a activity, respectivelye.  The variable names are slightly modified from the "features.txt" input file content, for example,  all parenthesis are removed. The row names merge the subject and the activity, for examplee, the row "1_LAYING" describes the means of the subject 1 for the LAYING activity.
